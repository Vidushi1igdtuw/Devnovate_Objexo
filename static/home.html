About Space Station Detector

Overview

The Space Station Object Detector leverages AI to identify critical equipment inside space stations — such as Toolboxes, Oxygen Cylinders, and Fire Extinguishers — ensuring astronaut safety and operational efficiency.

Key Features

  • Real-time object detection using YOLOv8 neural network
  • Interactive prediction page with visual results
  • Detection history storage with scrollable logs
  • Lightweight, responsive space-themed web UI

How It Works

1. Dataset Preparation: Synthetic dataset with space equipment under varied lighting and angles.

2. Model Training: YOLOv8 trained for 50 epochs to balance speed and accuracy.

3. Prediction: Uploaded images processed to identify and count equipment instantly.

4. History Logging: Predictions stored for performance analysis and mission tracking.
